Keyboard Shortcuts for Screen Reader Users

  • Tab / Shift + Tab - Move to next/previous interactive element
  • Enter - Activate a link or button
  • H - Jump to next heading (in NVDA/JAWS)
  • D - Jump to next landmark region
  • Ctrl + + / Ctrl + - - Increase / decrease text zoom
  • Insert + F7 - List of links (NVDA/JAWS)
  • Insert + F6 - List of headings
